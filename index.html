About Me

Ike Bischof

I do web development, videos, and animation. I make interactive web projects, which sometimes includes games, and I have a YouTube channel called Ikebot. I like to use coding to solve practical problems like automating tedious work, and impractical problems, like finding the best method to store the maximum possible amount of information on a cheat sheet for a test. These projects are an opportunity to learn and talk about interesting topics like information theory, game design, or how computers store data.

I'm currently a senior at Sacramento State University. I live in Roseville, California.
